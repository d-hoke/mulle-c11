# mulle-c11

A C header that enables more feature-rich compilation across various compilers
and platforms.

Used in various other mulle-projects.

## Install

Install the prerequisites first:

| Prerequisites                                 |
|-----------------------------------------------|
| [cmake](cmake.org)                            |

```
mkdir build 2> /dev/null
(
   cd build ;
   cmake .. ;
   make install
)
```
